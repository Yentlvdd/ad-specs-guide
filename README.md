# Digital Ad Specs Guide

A modern, responsive reference guide for Digital Ad Specifications built with React and Tailwind CSS.

## Features

- **Multi-Platform Support**: Meta, TikTok, LinkedIn, Pinterest, YouTube, Google Display.
- **Visual Reference**: Ad cards show the actual aspect ratio of the ad format.
- **Quick Copy**: One-click copy for dimensions.
- **Responsive Design**: Works on desktop and mobile.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React
- Tailwind CSS
- Lucide React (Icons)
- Vite
